# BD2_Projekt

## Informacje o projekcie zaliczeniowym

Projekt zaliczeniowy z przedmiotu **Bazy Danych 2 (BD2)** składa się z dwóch obowiązkowych elementów:

1. **Działający kod źródłowy** – umieszczony w repozytorium GitHub/GitLab.  
   Kod powinien być kompletny, uruchamialny i zgodny z założeniami tematu projektu.

2. **Raport projektowy** – dołączony do repozytorium w formacie `.pdf`.  
   Raport powinien zawierać:
   - opis problematyki projektu,  
   - zastosowaną technologię,  
   - zwięzły opis działania aplikacji,  
   - prezentację wyników działania aplikacji,  
   - odniesienia do źródeł zewnętrznych (bibliografia, dokumentacja, artykuły itp.).

---
## Praca zespołowa

Jeżeli projekt realizowany jest **w zespole 2 lub 3 osób**, obowiązuje następująca zasada:

> Każdy członek zespołu **musi posiadać własne repozytorium** z kopią projektu (kodem i raportem).  
> Repozytoria powinny być spójne i zawierać te same wersje plików.
